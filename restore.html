回復購買（Restore Purchases）

AlgoPro｜最後更新：2026-02-07

「回復購買（Restore Purchases）」用於重新驗證你已購買／已訂閱的狀態， 例如：更換裝置、重新安裝 App、或你發現訂閱權益未正確解鎖。

回復購買不會重新收費。它只會向 Apple 查詢並同步你現有的購買／訂閱狀態。

在 App 內回復購買

  1. 打開 AlgoPro
  2. 前往「訂閱／解鎖」頁面
  3. 按 「回復購買（Restore Purchases）」
  4. 確保裝置已連線到互聯網
  5. 按需要完成 Apple ID 驗證（例如 Face ID／Touch ID／密碼），以便 Apple 驗證你的購買／訂閱狀態

提示：如你曾用另一個 Apple ID 購買／訂閱，必須登入同一個 Apple ID 才可成功回復。

常見問題（FAQ）

  • 按咗回復購買都未解鎖？
    請先確認：1) 有網絡 2) Apple ID 與當初購買／訂閱一致 3) 再試一次回復購買。
  • 訂閱已到期／已取消，點解回復唔到？
    呢個係正常情況：回復購買只會同步「目前有效」的購買／訂閱狀態；如訂閱已到期或已取消，權益會按狀態停止。
  • 已退款點算？
    若 Apple 已處理退款，相關購買／訂閱狀態可能會變為無效，回復購買亦不會重新啟用權益——呢個亦屬正常情況。
  • 會唔會因為按回復購買而再次扣錢？
    不會。回復購買只作重新驗證與同步狀態，不會產生新收費。
